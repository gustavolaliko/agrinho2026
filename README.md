# EcoAgro Monitor

Projeto desenvolvido para o Concurso Agrinho 2026.

## Tema
Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Objetivo
Auxiliar produtores rurais a monitorar recursos naturais e incentivar práticas sustentáveis.

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Controle do consumo de água
- Registro de produção
- Indicador sustentável
- Relatórios simples
